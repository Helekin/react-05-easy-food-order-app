## Getting Started

### Prerequisites

- Use any code editor

### Installation

1. Clone the repo

```sh
git clone https://github.com/Helekin/react-05-easy-food-order-app.git
```

2. Create your .env file and use your credentials (optional)

3. Runs the app in the development mode

```sh
npm run start
```
